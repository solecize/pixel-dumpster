# kebab-style-naming

All project files and folders should use kebab-case naming, unless a tool requires a specific name (for example: `CMakeLists.txt`, `sdkconfig.defaults`, or `idf.py`).

## Examples

- `documentation/pixel-dumpster.md`
- `examples/python-client.py`
- `components/pd-network/pd-network.c`
- `components/pd-storage/include/pd-storage.h`

## Exceptions

- ESP-IDF build files (`CMakeLists.txt`, `sdkconfig.defaults`)
- Tooling defaults (e.g., `idf.py` configuration outputs)
